## 0.1.0 - 2026-04-29

### Добавлено

- Добавлен этот журнал изменений.
- В README добавлен быстрый маршрут со ссылками на конкретные шаги.
- В инструкции по сертификатам описаны три варианта: тестовый самоподписанный сертификат, сертификат предприятия и автоматический HTTPS Caddy.
- Добавлены проверки документации для локального запуска и GitHub Actions.
- Добавлена проверка внешних ссылок и бейдж GitHub Actions для README.
- Добавлена MIT-лицензия.

### Изменено

- README сокращён до практической пошаговой инструкции.
- Описание переменных перенесено в первый шаг и сформулировано как действие.
- `.env.example` упрощён: оставлены только переменные, которые нужны текущему шаблону.
- В шаге запуска уточнена первая миграция: сначала надо пересоздать службу приложения, чтобы освободить старый HTTP-порт, затем запускать Caddy.
- В README уточнено, что шаги после поиска службы выполняются из каталога приложения, а файлы шаблона надо скопировать рядом с основным Compose-файлом.
- В шаге 1 уточнено, откуда брать `.env.example` и куда помещать `.env`.
- Быстрый маршрут переписан как короткая карта действий с пояснениями.
- Из раздела о содержимом шаблона убраны стандартные файлы репозитория, не участвующие в шагах настройки.
- Заголовок быстрого маршрута заменён на более разговорный.

### Удалено

- Удалены OpenSSL-шаблоны из `certs/openssl/`; сертификаты теперь описаны через команды и внешние инструкции.
- Удалены подробные разделы про проверку цепочки, подготовку репозитория к публикации и сравнение Caddy с Nginx.
